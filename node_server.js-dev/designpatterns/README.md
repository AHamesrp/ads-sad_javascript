---

### 🔵 **Singleton**

* **O que é:** Garante que exista **apenas uma instância** de uma classe.
* **Usado para:** Banco de dados, configurações, logs.

---

### 🟩 **Adapter**

* **O que é:** **Adapta interfaces incompatíveis**, funcionando como um **tradutor** entre classes.
* **Usado para:** Integrar sistemas antigos com novos ou bibliotecas diferentes.

---

### 🟦 **Observer**

* **O que é:** Permite que **vários objetos sejam notificados** quando outro objeto muda.
* **Usado para:** Sistemas de eventos, GUIs, notificações em tempo real.
